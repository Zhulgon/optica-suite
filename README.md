# Optica Suite

Aplicacion web para gestion comercial y operativa de opticas.

## Stack

- Backend: NestJS + Prisma + PostgreSQL
- Frontend: React + Vite + TypeScript
- Monorepo: pnpm workspaces
- Infra local: Docker Compose

## Inicio rápido

Requisitos:

- Node.js 20+
- pnpm
- Docker

Instalacion:

```bash
pnpm install
```
