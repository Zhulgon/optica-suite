# Guion de Demo Comercial (30 minutos)

Objetivo: mostrar valor de negocio, no solo pantallas.

## Min 0-3: Contexto

1. Pregunta dolor principal:
   - desorden de historias?
   - falta de trazabilidad?
   - control de ventas?
2. Confirma objetivo de la demo.

## Min 3-8: Pacientes + Historia clinica

1. Crear paciente nuevo.
2. Abrir modulo de historias.
3. Registrar una historia rapida.
4. Mostrar que queda consultable e imprimible.

Mensaje clave: menos papel suelto, mas orden medico.

## Min 8-13: Agenda

1. Programar cita con optometra.
2. Mostrar filtros por fecha/estado/profesional.
3. Cambiar estado de cita.

Mensaje clave: mejor control de agenda y menos perdida de seguimiento.

## Min 13-20: Venta completa

1. Registrar venta con paciente.
2. Mostrar montura y lente.
3. Mostrar vendedor responsable y utilidad estimada.
4. Mostrar ventas recientes.

Mensaje clave: cada venta queda trazable y medible.

## Min 20-25: Auditoria y reportes

1. Abrir auditoria.
2. Filtrar y mostrar acciones clave.
3. Abrir reportes operativos/comerciales.

Mensaje clave: el admin controla que pasa y quien hizo cada accion.

## Min 25-30: Cierre

1. Recapitular beneficios en lenguaje de negocio.
2. Proponer piloto de 15-30 dias.
3. Explicar plan y precio recomendado.
4. Agendar fecha de decision.

## Tips de conversion

1. Hablar en resultados: tiempo, control, menos errores.
2. Usar datos reales del cliente en la demo cuando sea posible.
3. Cerrar siempre con siguiente paso concreto (fecha y responsable).
