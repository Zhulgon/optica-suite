# SLA de Soporte (Base)

## Niveles de severidad

| Severidad | Definicion | Tiempo objetivo primera respuesta |
|---|---|---|
| Critica | Sistema caido o venta bloqueada total | <= 1 hora habil |
| Alta | Modulo clave con falla parcial (agenda, historias, ventas) | <= 4 horas habiles |
| Media | Error no bloqueante o inconsistencia visual | <= 1 dia habil |
| Baja | Mejora o duda funcional | <= 2 dias habiles |

## Horario de soporte

- Lunes a viernes: 8:00 a.m. - 6:00 p.m.
- Sabados: 8:00 a.m. - 12:00 m.
- Domingos/festivos: guardia solo para incidentes criticos (si aplica plan).

## Canales

1. WhatsApp de soporte.
2. Correo de soporte.
3. Repositorio de incidencias (cuando aplique).

## Alcance incluido

- Incidentes de plataforma.
- Ayuda funcional de uso.
- Ajustes menores de configuracion.
- Actualizaciones de mantenimiento.

## Alcance no incluido

- Nuevas funcionalidades a medida.
- Integraciones externas no pactadas.
- Soporte de infraestructura de terceros no administrada por proveedor.

## Requisitos del cliente

1. Designar un contacto operativo.
2. Reportar incidente con evidencia (captura, hora, modulo).
3. Mantener backups y buenas practicas acordadas.
