# Proceso de Venta y Onboarding

## 1) Flujo comercial recomendado

1. Diagnostico (30-45 min): entender operacion actual.
2. Demo guiada (30 min): mostrar flujo real de paciente a venta.
3. Piloto controlado (15-30 dias): una sede y usuarios clave.
4. Cierre comercial: firma y pago inicial.
5. Implementacion formal: configuracion, migracion, capacitacion.
6. Go-live + seguimiento: 2 semanas de acompanamiento.

## 2) Checklist de preventa

- Tipo de cliente (pequena, mediana, multi-sede).
- Numero de usuarios por rol.
- Volumen aproximado de pacientes e historias.
- Necesidad de migracion de datos historicos.
- Modalidad objetivo (SaaS o LAN).

## 3) Onboarding tecnico estandar

1. Crear sedes.
2. Crear usuarios por rol (admin, optometra, asesor).
3. Cargar monturas base.
4. Validar login + permisos por rol.
5. Validar flujos clave: pacientes, historias, agenda, ventas.
6. Activar backups y monitoreo.

## 4) Capacitacion sugerida

Sesion 1 (admins):

- roles y permisos
- seguridad
- auditoria y reportes

Sesion 2 (equipo operativo):

- pacientes
- historias clinicas
- agenda
- ventas y cierre diario

## 5) Criterios de salida en vivo

1. Usuario admin activo y con 2FA.
2. Al menos un optometra y un asesor creados.
3. Backups probados.
4. Cliente completa 3 casos reales de punta a punta sin bloqueo.
